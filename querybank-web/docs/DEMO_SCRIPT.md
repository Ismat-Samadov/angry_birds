## 🚨 Troubleshooting During Demo

### If AI takes too long (>10 seconds):
**Say**: "Gemini API yüksək yükdədir. Bir daha cəhd edək..."
**Action**: Click another predefined question

### If you get "503 Service Unavailable":
**Say**: "Bu Google-un API limit-idir. Production-da rate limit yüksək olar."
**Action**: Wait 30 seconds and try again

### If login doesn't work:
**Check**:
- Correct URL (localhost:3001 or Vercel URL)
- Credentials: demo@querybank.az / demo123
- Dev server is running

### If stats don't load:
**Check**:
- Database connection (DATABASE_URL)
- Network connection
- Refresh page
